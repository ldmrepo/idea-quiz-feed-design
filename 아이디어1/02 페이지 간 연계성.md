  - 추천 퀴즈 표시
  - 학습 목표 현황
- 소셜 섹션 진입
  - 친구 활동 피드
  - 실시간 배틀 현황
  - 그룹 알림
- 개인화 섹션 진입

  - 주요 분석 지표
  - 개인 알림

2. 학습 섹션 연계

```

└─ AI 어시스턴트 (문제 추천)
└─ 커뮤니티 (문제 공유)

복습 시스템
└─ AI 어시스턴트 (일정 최적화)
└─ 학습 분석 (효율성 측정)
└─ 진도 관리 (복습 현황)

학습 진도
└─ 대시보드 (목표 대비)
└─ 분석 리포트 (진도율)
└─ 스터디 그룹 (그룹 진도)
```

3. 소셜 섹션 연계

```

└─ 스터디 그룹 (모집/홍보)
└─ 배틀 시스템 (대회 소식)

스터디 그룹 페이지
└─ 퀴즈/문제 (그룹 학습)
└─ 학습 분석 (그룹 통계)
└─ AI 어시스턴트 (활동 추천)
```

4. 개인화 섹션 연계

```

└─ 대시보드 (주요 지표)

└─ 진도 관리 (추세 분석)
```
