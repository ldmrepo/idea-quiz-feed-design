```markdown

- 연속 선택형 (Continuous Choice)
- 리듬 기반 선택형 (Rhythm-based)
- 반응 속도형 (Reaction Time)
```

2. 난이도 진화형 추가:

```markdown
진화형 선택형
```

4. 메타인지 강화형:


